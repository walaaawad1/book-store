//Al-Mureeh for Sudanese Secondary Certificate

A modern digital platform for selling books and study aids for the Sudanese Secondary Certificate, with AI features to identify books from their cover images. This project is front-end only and does not include a back-end.

//live demo:https://bookstorewalaaa.netlify.app/

//Features:

Browse book catalog by track (Science, Literary).

Detailed page for each book with price, description, and cover image.

User-friendly interface with Arabic support and right-to-left (RTL) layout.

AI-powered identification of book subjects using Google Gemini API from cover images.

Responsive modern design with Tailwind CSS.

Smooth animations and transitions using Framer Motion.

Customizable icons using Lucide React.

// Technologies Used:

React 

TypeScript

Vite (for fast development and build)

Tailwind CSS (for UI styling)

Framer Motion (for animations)

Lucide React (for icons)

Google Gemini API (for book subject recognition from images)

//Installation & Running

Install dependencies:

npm install

Create a .env file in the project root with your Gemini API key:

GEMINI_API_KEY=YOUR_REAL_API_KEY

Run the project locally:

npm run dev

Open your browser at the displayed address (usually http://localhost:5173).

//Notes:

This project is front-end only. No back-end or payment processing is included.

Gemini API is used for testing book identification from images.

The project can be extended later to add a back-end for users and orders.
